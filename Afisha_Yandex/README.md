# Решение аналитических задач для продуктового отдела сервиса Яндекс Афиша. 

## Описание проекта
Представлены данными сервиса Яндекс Афиша. С его помощью пользователи могут узнавать информацию о мероприятиях 
в разных городах и покупать на них билеты. Сервис сотрудничает с партнёрами — организаторами мероприятий и билетными операторами, 
которые предоставляют информацию о событиях и выставляют билеты на продажу.  

## Структура проекта
### Проект состоит из двух частей:
1. Анализ данных с помощью SQL и создание дашборда в Yandex DataLens включая, рассчёт ключевых метрик и основных показателей продукта, а также построение аналитического дашборда в DataLens.

2. Исследовательский анализ данных и проверка гипотез с помощью Python.
